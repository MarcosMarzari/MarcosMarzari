## 🎮 Projetos em Destaque

Aqui estão alguns dos projetos mais desafiadores e divertidos que desenvolvi recentemente:

### ⚔️ [Beast Hunter RPG (Open World)](link-do-seu-repositorio)
Um RPG de exploração em mundo aberto rodando 100% no navegador.
- **Destaques Técnicos:** Renderização gráfica 2D em tempo real com **Canvas API**, sistema de colisão por matriz estruturada, controle D-Pad responsivo para dispositivos móveis e mecânica de combate por turnos com tabelas de crescimento de atributos por classe.
- **Tecnologias:** JavaScript Puro (ES6+), HTML5 Canvas e CSS Moderno.

### 📊 [Simulador Financeiro Inteligente](link-do-seu-repositorio)
Uma ferramenta interativa para planejamento financeiro e simulações de tributação.
- **Destaques Técnicos:** Lógica condicional avançada para cálculo de impostos (perfil CLT vs. Autônomo), design responsivo focado em mobile (*Mobile First*) e interface acessível com accordions nativos para FAQs.
- **Tecnologias:** HTML, CSS (Flexbox/Grid), JavaScript.
